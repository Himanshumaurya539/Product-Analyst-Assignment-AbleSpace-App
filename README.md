# Product-Analyst-Assignment-AbleSpace-App
This project focuses on analyzing user behavior and improving feature effectiveness within the goal-setting flow of the AbleSpace application.

🔍 Overview
The primary objective is to understand how users interact with the goal creation and tracking system, identify friction points, and propose data-driven improvements to enhance engagement and usability.

🧭 User Journey Mapping
The project maps the end-to-end flow of a user creating a goal:
User signs up with basic details (name, email, role, password)
Lands on the dashboard
Navigates to the Goals section via sidebar
Clicks on Add Goal
Enters goal title and description
Saves the goal successfully

❓ Key Hypotheses
How frequently users check goal progress reports
Which parts of the goal report provide the most value
How often student goal reports are updated

📈 Insights & Analysis Plan
Analyze conversion from sign-up to goal creation
Track user engagement with goals and reports
Measure goal completion rates and timelines
Identify drop-offs and underutilized features
Evaluate user interaction patterns across sections
Key metric: Number of users creating more than 4 goals within a given period

💡 Product Suggestions
Introduce goal reminders to improve completion rates
Add interactive guides/onboarding to reduce user confusion
Optimize UI/UX based on usage patterns and inactive sections

🎯 Outcome
This analysis helps:
Identify usability gaps
Improve feature adoption
Enhance user engagement and productivity
Support data-driven product decisions
